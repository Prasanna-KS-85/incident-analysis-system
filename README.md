# Explainable Multi-Agent Decision Intelligence Framework for Context-Aware Citizen Grievance Prioritization (TARP)

An Explainable, Multi-Agent Decision Intelligence System for prioritizing citizen grievances.

## Key Novelty
- **Complaint Criticality Index (CCI)**: A novel metric for assessing the urgency and importance of grievances.
- **Multi-Agent Architecture**: A system of specialized agents working together to classify, analyze, and prioritize complaints.
